Endpoints

POST /register

Registra um novo usuário

Body: { "username": "string", "password": "string" }

Retorno: Redireciona para /login

POST /login

Autenticação do usuário

Form Data: username, password

Retorno: { "access_token": "string", "token_type": "bearer" }

GET /check-auth

Verifica autenticação

Header: Authorization: Bearer {token}

Retorno: { "authenticated": true }

POST /upload

Upload de arquivo de áudio para conversão

Requer: 
- Arquivo de áudio (FormData)
- Header: Authorization: Bearer {token}

Retorno: { "message": "string", "filename": "string" }

POST /apply-scents/{ad_id}

Aplica efeitos Scents ao anúncio

Path: ad_id (número)
Header: Authorization: Bearer {token}

Retorno: { "message": "string", "ad_id": number }

GET /ad/{ad_id}

Obtém informações de um anúncio

Path: ad_id (número)

Retorno: { "ad_id": number, "filename": "string", "scents_applied": boolean }

GET /stats/{ad_id}

Obtém estatísticas de um anúncio

Path: ad_id (número)

Retorno: { "ad_id": number, "views": number }
